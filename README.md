# GraphPaper — Feedback

This is the public feedback tracker for **GraphPaper**, a browser-based
graph-paper / drawing canvas.

Found a bug? Have a feature idea? **[Open an issue →](https://github.com/Windovvsill/graphpaper-feedback/issues/new/choose)**

- 🐛 **Bug reports** — what happened, what you expected, and how to reproduce it.
- 💡 **Feature ideas** — what you'd like to be able to do, and why.

Browse [existing issues](https://github.com/Windovvsill/graphpaper-feedback/issues)
first — a 👍 reaction on one that matches helps us prioritise, and saves you
writing a duplicate.

> This repo holds **no source code** — it exists only to collect feedback. The
> app itself lives in a separate (private) repository.
